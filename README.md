PCAPScan
--------

PCAPscan analyzes a large amount of pcap files by extracting interesting
information

Requirements
============

* python3
* make
* see requirements.txt

Installation
============

There is a make file that helps installing the necessary packets in a
python environment:

```
% make install
```

For more dev utilities see

```
% make help
```
